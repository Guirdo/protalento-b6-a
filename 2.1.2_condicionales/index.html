Cadenas de texto.

Las cadenas de texto se pueden concatenar. Concatenar -> unir valores

  1. "texto" + "texto" => textotexto
  2. variable = "helado"; variable1 = 2; `${variable} y otro valor ${variable1}.` => helado y otro valor 2
